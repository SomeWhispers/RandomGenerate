# RandomGenerate

## このアプリの詳細
C#を用いた乱数生成アプリ
2つの数値x,yを与え、x,y の範囲は0から1の間とする。
xがyより大きければ分類ラベル:1、xがyより小さければ分類ラベル:0 というようにそれぞれの組み合わせにラベルを付与する。
